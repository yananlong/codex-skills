# PRISMA 2020 Core Checklist (Operational)

Use this checklist to ensure minimum systematic-review reporting quality.

## Protocol and scope

- State the review question clearly.
- State inclusion and exclusion criteria.
- State search sources and date range.
- State screening process and decision rules.

## Discovery and selection

- Record exact query strings.
- Record retrieval dates.
- Record deduplication method.
- Record screening counts for all flow stages.

## Data extraction and synthesis

- Record extraction fields consistently.
- Separate primary and secondary outcomes.
- Distinguish descriptive summary from causal inference.
- Report inconsistency and heterogeneity explicitly.

## Bias and confidence

- Assess risk of bias per study.
- Address publication and reporting bias.
- Grade overall confidence with rationale.
- State key limitations and unresolved uncertainties.

## Reproducibility

- Keep protocol, logs, and evidence table aligned.
- Preserve auditable links from conclusions to included studies.
- Report deviations from protocol and rationale.
