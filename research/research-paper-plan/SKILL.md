# Research Paper Plan

## Quick start

1. List the claims the paper wants the reader to accept.
2. Build a claims-to-evidence matrix before outlining sections.
3. Map each claim to figures, tables, experiments, citations, and limitations.
4. Outline the paper only after the evidence map is coherent.

## Workflow

### 1) Start from claims, not section titles

- Write the primary contribution claim, supporting claims, and limitation claims.
- Check that each claim has a clear evidence path.
- Downgrade or remove claims that lack support.

### 2) Build the evidence map

- Use `references/claims-evidence-matrix-template.md`.
- Record which artifact supports each claim and which section should carry it.
- Note missing evidence early so the paper plan can drive research, not just describe it.

### 3) Outline the manuscript

- Use `references/paper-outline-template.md`.
- Keep motivation, method, evaluation, limitations, and failure modes aligned with the actual evidence.
- Treat threats to validity as first-class content, not cleanup text.

### 4) Plan figures and tables deliberately

- Every figure and table should answer a paper question.
- Avoid decorative results that do not advance the main argument.
- Mark which exhibits are mandatory for acceptance and which are optional.
