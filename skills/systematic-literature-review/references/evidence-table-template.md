## Extraction guidance

- Use one row per included study.
- Prefer citing the peer-reviewed published version when available; use preprint links only as supplemental full-text access.
- Keep outcome labels consistent across rows.
- Capture numeric effect sizes where available.
- Mark uncertainty explicitly instead of inferring unsupported values.
